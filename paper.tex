\section*{Introduction}

Phillip M. Parker, the most well-known author of 200.000 computer-generated books. He patented a method to generate a set of similar books from database. Parker's algorithm can be seen as so-called text-to-text generation methods. It takes existing text as the input and automatically produced a new text as output.
